## What Are Skills?

Skills are specialized procedures stored in `.claude/skills/*/SKILL.md`.
They provide domain-specific instructions that load on demand — only when
the relevant task is triggered.

## Adding a Skill

1. Create a directory: `.claude/skills/{skill-name}/`
2. Add a `SKILL.md` file with frontmatter and procedures
3. Register it in this file
4. Reference the trigger phrase in `CLAUDE.md` if needed
